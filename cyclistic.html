Chicago Bike Share Analysis

Analyzing historical trip data to inform a marketing strategy to convert casual users to members

This is the final case study in Google's Data Analytics Certificate. The aim is to help Cyclistic, a fictitious bike-share company based in Chicago, develop a marketing strategy to sell annual memberships to "casuals" (single-ride and day-pass users). In order to do so, you are provided historical trip data which you are asked to analyze for differences between the casuals and members.

I documented my analysis thoroughly in two R Markdown produced HTML documents. Part I covers data processing and prepares the data for the actual analysis in Part II. Furthermore, I created a PowerPoint presentation which gives a more practical high level overview including final recommendations for stakeholders. You can download it as a PowerPoint or PDF file. Finally, you can can find the GitHub repository containing the entire project here.
